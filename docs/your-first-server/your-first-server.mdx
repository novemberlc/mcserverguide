So you want to run a Minecraft server, but you don't know how to set it up. It's actually pretty easy. In the following guide, you'll decide what you want to run your server on, learn how to configure it, and learn how to allow others to join. This guide will get you up and running with a basic Minecraft server. Head to the next section at the bottom of this page to start!
